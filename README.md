NextJs Api Routes

### Prisma

#### 连接数据库

1. 在.env 文件中添加本地数据库地址
2. 部署到你的数据库

```bash
    yarn prisma migrate  dev
    yarn prisma migrate  deploy
    yarn prisma generate
```
